# Product Backlog – Student Task Tracker

## User Stories

1. As a student, I want to create an account so that I can manage my tasks  
Priority: High | Points: 5  
Acceptance Criteria:
- User can input email and password  
- Account is saved in system  
- User receives confirmation  

2. As a student, I want to log in so that I can access my tasks  
Priority: High | Points: 5  
Acceptance Criteria:
- Valid login redirects to dashboard  
- Invalid login shows error  

3. As a student, I want to add tasks so that I can track assignments  
Priority: High | Points: 8  
Acceptance Criteria:
- Task title can be entered  
- Task is saved  
- Task appears in list  

4. As a student, I want to edit tasks so that I can update details  
Priority: Medium | Points: 5  
Acceptance Criteria:
- Task can be modified  
- Changes are saved  

5. As a student, I want to delete tasks so that I can remove completed ones  
Priority: Medium | Points: 3  
Acceptance Criteria:
- Task can be deleted  
- Task disappears from list  

6. As a student, I want to set deadlines so that I can track due dates  
Priority: High | Points: 5  
Acceptance Criteria:
- Deadline can be selected  
- Deadline is displayed  

7. As a student, I want to mark tasks as complete so that I know progress  
Priority: High | Points: 3  
Acceptance Criteria:
- Task can be marked complete  
- Status is updated  

8. As a student, I want to view all tasks in a dashboard so that I see everything  
Priority: High | Points: 8  
Acceptance Criteria:
- Tasks are listed  
- Completed tasks are labeled  

9. As a student, I want reminders so that I don’t miss deadlines  
Priority: Low | Points: 8  
Acceptance Criteria:
- Notification is triggered  
- Reminder shows before deadline  

10. As a student, I want to log out so that my account is secure  
Priority: High | Points: 2  
Acceptance Criteria:
- Logout button works  
- User is redirected
