# QA Plan – Student Task Tracker

## Test Levels
- Unit Testing: Test individual functions
- Integration Testing: Test combined components
- System Testing: Test full application

## Entry Criteria
- Code is completed
- Features are implemented

## Exit Criteria
- All tests pass
- No critical bugs (S1)

## Severity Levels
- S1: Critical (system unusable)
- S2: Major (feature broken)
- S3: Minor (small issue)
- S4: Low (cosmetic)
